# 📝 MD Viewer

옵시디언 스타일 마크다운 뷰어 / 에디터입니다.

> 이 홈 화면은 앱 폴더의 **`home.md`** 파일입니다. 이 파일을 편집하면 홈 화면이 바뀝니다. (코드 수정·재빌드 불필요, 앱만 다시 실행)

## 단축키

| 기능 | 단축키 |
|------|--------|
| 편집 / 분할 / 미리보기 | `Ctrl+1` / `Ctrl+2` / `Ctrl+3` |
| 검색 / 폴더 전체 검색 / 바꾸기 | `Ctrl+F` / `Ctrl+Shift+F` / `Ctrl+H` |
| 커맨드 팔레트 | `Ctrl+P` |
| 파일 열기 / 폴더 추가 | `Ctrl+O` / `Ctrl+Shift+O` |
| 저장 / 탭 닫기 | `Ctrl+S` / `Ctrl+W` |
| 설정 | `Ctrl+,` |
| 글꼴 크기 | `Ctrl + +` / `Ctrl + -` / `Ctrl + 휠` |
| 사이드바 토글 | `Ctrl+B` |
| Claude 패널 | `Ctrl+J` |
| 되돌리기 / 다시 실행 | `Ctrl+Z` / `Ctrl+Shift+Z` |

## 기능

- **탭** 으로 여러 문서를 동시에 (가운데 휠클릭으로 닫기, 탭 이름 더블클릭으로 이름 변경)
- **여러 폴더(Vault)** 추가, 각각 접기/펴기
- **편집기 문법 색상** + **분할 동기 스크롤**
- **두 파일 분할 편집**: 상단 탭이나 왼쪽 탐색기 파일을 오른쪽으로 끌어다 놓으면 서로 다른 파일을 나란히 편집
- **목차** 패널, 위키링크 `[[노트]]`, 자동 저장(설정)
- **테마 세트**: 다크 / 라이트 / 노르드 / 솔라라이즈드
- **파일 형식**: `.md` 📝 · `.txt` 📄 · `.json` 🔧 · `.html` 🌐
- **HTML 듀얼 뷰**: 분할 모드에서 소스 + 실제 렌더링 페이지 동시 보기
- **Claude 연동**: `Ctrl+J` 로 노트에 대해 질문·요약·수정 (반영하기 버튼)

```js
function hello(name) {
  console.log(`안녕하세요, ${name}님!`);
}
```

> 왼쪽 위 **＋📂** 로 폴더를 추가하거나, 파일을 창에 끌어다 놓아 시작하세요.
