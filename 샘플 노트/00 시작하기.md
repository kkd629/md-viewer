# 🏠 시작하기

이 폴더는 **MD Viewer** 기능을 둘러보기 위한 샘플 보관함(Vault)입니다.
왼쪽 사이드바에서 파일을 눌러 이동하거나, 아래 위키링크를 클릭해 보세요.

## 노트 목록

- [[01 마크다운 문법]] — 마크다운 기본 문법 총정리
- [[02 코드 하이라이트]] — 여러 언어 코드 블록 예시
- [[03 표와 체크리스트]] — 표 · 할 일 목록 · 인용
- [[04 회의록 예시]] — 실제 사용 예시 (회의록)

## 이렇게 해보세요

1. `Ctrl + 2` 를 눌러 **분할 모드**로 전환 → 왼쪽에서 편집하면 오른쪽 미리보기가 같이 스크롤됩니다.
2. `Ctrl + Shift + F` 로 이 폴더 전체에서 **"마크다운"** 을 검색해 보세요.
3. 오른쪽 위 ⚙ 또는 `Ctrl + ,` 로 **테마**를 라이트/노르드로 바꿔 보세요.
4. 왼쪽 **목차** 패널에서 제목을 클릭하면 해당 위치로 이동합니다.

> 💡 위키링크 `[[노트 이름]]` 은 이 폴더 안에서 같은 이름의 파일을 찾아 열어 줍니다.
